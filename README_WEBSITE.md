# Stargazer Project Page

This folder is a static project page scaffold for the Stargazer paper.

## Files to update before publishing

- `index.html`: replace the top button links with the final paper, code, and dataset URLs.
- `static/pdfs/`: swap or add final paper figures as needed.
- `static/images/stargazer-mark.svg`: replace with an official logo if desired.

## Deploy on GitHub Pages

1. Push this folder to a repository.
2. Make sure `index.html` is at the publishing root.
3. Enable GitHub Pages for the branch/folder.
4. Keep `.nojekyll` so the static paths are served without Jekyll processing.
